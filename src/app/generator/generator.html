generator-gulp-angular

I created a Yeoman generator for scaffolding an Angular application using Gulp as build tool.

It becomes quite big, I welcomed @zckrs as co-owner and it represent since the major part of my open source contribution.

The point of the project is having a tool to scaffold a new frontend project with all the state of the art development and optimization process only assuming the us of Angular and Gulp but letting the user choose it's CSS preprocessor (such as Sass or Less), JS preprocessor (such as Coffee or Babel) or it's UI framework (such as Bootstrap or Angular Material) for example.

This website has bean scaffold with it!
